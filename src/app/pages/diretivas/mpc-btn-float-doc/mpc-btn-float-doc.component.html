MPC Btn Float

Descrição: O mpcBtnFloat do MpcComponents é uma diretiva que transforma qualquer elemento <button> em um botão flutuante fixo, aplicando estilos CSS padrão e permitindo a inclusão de ícones personalizados.
Propriedades da Diretiva:
  • icone: Classes CSS do ícone a ser exibido (ex: 'bi bi-arrow-up-short').
Estilos CSS Aplicados Automaticamente:
  • position: fixed (posicionamento fixo na tela)
  • right: 15px (distância da borda direita)
  • bottom: 15px (distância da borda inferior)
  • z-index: 998 (camada de sobreposição)
  • width/height: 44px (tamanho do botão)
  • border-radius: 50px (formato circular)
  • display: flex (layout flexível para centralizar ícone)
  • transition: all ease-in 0.4s (transições suaves)
  • box-shadow: 0 0 10px rgba(0, 0, 0, 0.2) (sombra)
Propriedades CSS do Elemento:
  • [style.background-color]: Cor personalizada para o fundo do botão (hex, rgb, rgba, nome da cor).
  • [style.color]: Cor personalizada para o ícone do botão (hex, rgb, rgba, nome da cor).
  • disabled: (opcional) Atributo que indica se o botão está desabilitado.
  • tabindex: (opcional) Índice de tabulação para acessibilidade.
  • aria-label: (opcional) Descrição do botão para acessibilidade.
Importação no arquivo .ts:
imports: [ MpcBtnFloatDirective ]
Exemplo de uso no arquivo .html:
<button mpcBtnFloat (click)="scrollToTop()" [style.background-color]="'#007bff'" [style.color]="'white'" [icone]="'bi bi-arrow-up-short'"></button>
Observações importantes:
  • A diretiva mpcBtnFloat adiciona automaticamente os estilos CSS necessários ao elemento.
  • O botão flutuante é sempre visível e posicionado no canto inferior direito da tela.
  • O ícone é inserido dinamicamente pelo Angular, substituindo qualquer conteúdo interno do botão.
  • As cores são definidas através das propriedades CSS [style.background-color] e [style.color].
  • O botão possui formato circular com 44px de largura e altura.
  • As cores podem ser definidas em qualquer formato CSS válido (hex, rgb, rgba, nomes de cores, variáveis CSS).
  • Para eventos de clique, utilize (click)="suaFuncao()".
  • A diretiva inclui suporte para acessibilidade com tabindex e aria-label.
  • O botão possui transições suaves para melhor experiência do usuário.
