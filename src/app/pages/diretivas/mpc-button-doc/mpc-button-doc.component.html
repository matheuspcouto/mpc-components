MPC Button

Descrição: O mpcButton do MpcComponents é uma diretiva que transforma qualquer elemento <button> em um botão estilizado, permitindo a inclusão de ícones e personalização via propriedades CSS.
Propriedades da Diretiva:
  • texto: O texto exibido no botão (obrigatório).
  • icone: (opcional) Ícone exibido no botão, utilizando classes do Bootstrap Icons ou outra biblioteca de ícones.
  • posicaoIcone: (opcional) Define a posição do ícone em relação ao texto: esquerda ou direita. Padrão: direita.
Propriedades CSS do Elemento:
  • [style.background-color]: Cor personalizada para o fundo do botão (hex, rgb, rgba, nome da cor).
  • [style.color]: Cor personalizada para o texto do botão (hex, rgb, rgba, nome da cor).
  • disabled: (opcional) Atributo que indica se o botão está desabilitado.
  • tabindex: (opcional) Índice de tabulação para acessibilidade.
  • aria-label: (opcional) Descrição do botão para acessibilidade.
Importação no arquivo .ts:
imports: [ MpcButtonDirective ]
Exemplo de uso no arquivo .html:
<button mpcButton (click)="minhaFuncao()" [style.background-color]="'#0E2749'" [style.color]="'white'" [icone]="'bi bi-code-slash'" [posicaoIcone]="'esquerda'" [texto]="'Exemplo de Botão'"></button>
Observações importantes:
  • A diretiva mpcButton adiciona automaticamente a classe CSS .mpc-btn ao elemento.
  • O texto e ícone são inseridos dinamicamente pelo Angular, substituindo qualquer conteúdo interno do botão.
  • As cores são definidas através das propriedades CSS [style.background-color] e [style.color].
  • A posição padrão do ícone é direita.
  • As cores podem ser definidas em qualquer formato CSS válido (hex, rgb, rgba, nomes de cores, variáveis CSS).
  • Para eventos de clique, utilize (click)="suaFuncao()".
  • A diretiva inclui suporte para acessibilidade com tabindex e aria-label.
