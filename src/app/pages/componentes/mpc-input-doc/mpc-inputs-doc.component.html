Descrição: Os mpc-input-* são componentes customizados para entrada de dados, implementando ControlValueAccessor para integração com formulários reativos. Todos os inputs suportam validação, acessibilidade e personalização via variáveis CSS.
Propriedades Comuns:
  • label: string (obrigatório) — Label do campo.
  • readonly: boolean (opcional) — Campo somente leitura.
  • disabled: boolean (opcional) — Campo desabilitado.
  • required: boolean (opcional) — Campo obrigatório.
  • id: string (opcional) — ID único do campo para acessibilidade.
  • tabIndex: number (opcional) — TabIndex para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
Propriedades Específicas:
  • min: number (opcional) — Valor mínimo (inputs de texto, número, área de texto).
  • max: number (opcional) — Valor máximo (inputs de texto, número, área de texto).
  • minDate: Date | string (opcional) — Data mínima (input de data).
  • maxDate: Date | string (opcional) — Data máxima (input de data).
  • options: any[] (opcional) — Opções para select/radio.
  • opcoes: any[] (opcional) — Opções para select (alias).
Instalação:
npm install mpc-lib-angular
Importação no arquivo .ts:

import { 
  MpcInputTextComponent,
  MpcInputNumberComponent,
  MpcInputDateComponent,
  MpcInputEmailComponent,
  MpcInputSenhaComponent,
  MpcInputTelefoneComponent,
  MpcInputCpfcnpjComponent,
  MpcInputBuscaCepComponent,
  MpcInputTextAreaComponent,
  MpcInputRadioComponent,
  MpcInputSelectComponent
} from 'mpc-lib-angular';

// No seu componente
imports: [
  MpcInputTextComponent,
  MpcInputNumberComponent,
  MpcInputDateComponent,
  MpcInputEmailComponent,
  MpcInputSenhaComponent,
  MpcInputTelefoneComponent,
  MpcInputCpfcnpjComponent,
  MpcInputBuscaCepComponent,
  MpcInputTextAreaComponent,
  MpcInputRadioComponent,
  MpcInputSelectComponent
]
Exemplo de uso no arquivo .html:
<!-- Input Text Básico -->
<mpc-input-text 
  label="Nome" 
  [(ngModel)]="nome" 
  [min]="3" 
  [required]="true">
</mpc-input-text>

<!-- Input Email com Validação -->
<mpc-input-email 
  label="Email" 
  [(ngModel)]="email" 
  [required]="true">
</mpc-input-email>

<!-- Input Senha com Validação -->
<mpc-input-senha 
  label="Senha" 
  [(ngModel)]="senha" 
  [required]="true">
</mpc-input-senha>

<!-- Input Select com Opções -->
<mpc-input-select 
  label="Estado" 
  [opcoes]="estados" 
  [(ngModel)]="estado">
</mpc-input-select>

<!-- Input com Acessibilidade -->
<mpc-input-text 
  label="Nome Completo"
  id="nome-completo"
  [tabIndex]="1"
  ariaLabel="Campo para inserir o nome completo da pessoa"
  [(ngModel)]="nome">
</mpc-input-text>
Personalização com Variáveis CSS:
/* Variáveis CSS comuns a todos os inputs */
--mpc-color-text: var(--mpc-color-primary);      /* Cor do texto do campo */
--mpc-color-error: #DC3545;                      /* Cor de erro */
--mpc-color-border: var(--mpc-color-tertiary);   /* Cor da borda do campo */
--mpc-color-border-success: var(--mpc-color-secondary); /* Cor da borda de sucesso */
--mpc-font-text: var(--mpc-font-default);       /* Fonte do campo */
