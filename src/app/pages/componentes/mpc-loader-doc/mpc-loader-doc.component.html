MPC Loader

O mpc-loader é um componente standalone que exibe uma animação de carregamento enquanto o conteúdo da página está sendo carregado. Utiliza signals para reatividade (Angular 17+) e não requer CommonModule.
Características:
  • Utiliza signals para reatividade
  • Controle via MpcLoaderService
  • Suporte a múltiplas chamadas simultâneas
  • Customização de cores via inputs
  • Controle automático de contador interno
Para utilizar o componente em uma página específica ou em situações que não sejam uma requisição HTTP, inclua o seguinte código no arquivo seu-component.ts
Importação no arquivo .ts: 
              
                ```
                imports: [ MpcLoaderComponent ]
                ```
Inicialização do componente e service: 
              
                ```
                private readonly mpcLoaderService = inject(MpcLoaderService);

                abrirLoading() {
                  this.mpcLoaderService.show();
                  // Implemente a lógica entre a abertura e o fechamento do loading
                  setTimeout(() => this.mpcLoaderService.hide(), 5000);
                }
                ```
Inclusão do componente no arquivo seu-component.html (se quiser utilizar na aplicação inteira, inclua no app.component.html): 
            
            ```
            <mpc-loader backgroundColor="#f5f5f5" color="#007bff"></mpc-loader>
            ```
Customização de cores: 
            
            ```
            <mpc-loader backgroundColor="#f5f5f5" color="#007bff"></mpc-loader>
            ```
Parâmetros de customização:
  • backgroundColor: Cor de fundo do loader (ex: "#f5f5f5", "rgba(0,0,0,0.5)", "transparent")
  • color: Cor do spinner do loader (ex: "#007bff", "#28a745", "#dc3545")
Métodos do MpcLoaderService:
  • show(): Exibe o loader (incrementa contador interno)
  • hide(): Oculta o loader (decrementa contador interno)
  • isLoading(): Retorna signal com estado atual do loader
