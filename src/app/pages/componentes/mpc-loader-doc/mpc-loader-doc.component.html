MPC Loader

Propriedades do Componente:
  • id: string (opcional) — ID do loader para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
O componente não possui outros @Input(). O controle de exibição é feito via MpcLoaderService.
Importação no arquivo .ts:

import { MpcLoaderComponent, MpcLoaderService } from 'mpc-lib-angular';

// No seu componente
imports: [MpcLoaderComponent]

// Injetar o serviço
private readonly mpcLoaderService = inject(MpcLoaderService);
Exemplo de uso no arquivo .html:
<!-- Loader Básico -->
<mpc-loader id="loader-principal"></mpc-loader>

<!-- Loader com Acessibilidade -->
<mpc-loader 
  id="loader-principal"
  [tabIndex]="0"
  ariaLabel="Carregamento da página">
</mpc-loader>
Exemplo de uso no arquivo .ts:
private readonly mpcLoaderService = inject(MpcLoaderService);

abrirLoading() {
  this.mpcLoaderService.show();
  // Implemente a lógica entre a abertura e o fechamento do loading
  setTimeout(() => this.mpcLoaderService.hide(), 5000);
}
Personalização com Variáveis CSS:
/* Cores do Loader */
--mpc-bg-loader: white;                    /* Cor de fundo do loader */
--mpc-color-loader: var(--mpc-color-primary); /* Cor do spinner do loader */
Métodos do MpcLoaderService:
  • show(): Exibe o loader (incrementa contador interno)
  • hide(): Oculta o loader (decrementa contador interno)
  • isLoading(): Retorna se o loader está carregando
