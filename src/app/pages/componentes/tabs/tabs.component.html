MPC Tabs

O mpc-tabs é um componente que permite criar tabs personalizados. Os tabs são utilizados para exibir informações de forma organizada e atraente. Abaixo, você pode ver como utilizar o componente e seus parâmetros.
  • tabs: Os tabs do componente. Formato: [{ id: 'string', titulo: 'string' }]
  • tabIndex: (opcional) O tabIndex do card.
  • ariaLabel: (opcional) O texto de acessibilidade do componente.
No arquivo seu-component.ts, importe o componente:
              
                ```
                imports: [ MpcTabsComponent ]
                ```
Ainda no seu-component.ts, crie um objeto que irá implementar a interface Tab do componente e irá popular as abas da sua página, e implemente um método de seleção de aba.
                
                  tabs: Tab[] = [
                  { id: 'conteudoTab1', titulo: 'Tab 1' },
                  { id: 'conteudoTab2', titulo: 'Tab 2' },
                  { id: 'conteudoTab3', titulo: 'Tab 3' }
                ]

                tabSelecionada: Tab = this.tabs[0];

                selectTab(tab: Tab) {
                  this.tabSelecionada = tab;
                }
No arquivo seu-component.html, adicione o seguinte código: 
                
                  <mpc-tabs [tabs]="tabs" [tabIndex]="0" ariaLabel="Tabs" (tabSelected)="selectTab($event)"></mpc-tabs>

                  <div *ngIf="tabSelecionada.id === 'conteudoTab1'" class="d-flex justify-content-center mt-3"> Conteúdo 1 </div>
                  <div *ngIf="tabSelecionada.id === 'conteudoTab2'" class="d-flex justify-content-center mt-3"> Conteúdo 2 </div>
                  <div *ngIf="tabSelecionada.id === 'conteudoTab3'" class="d-flex justify-content-center mt-3"> Conteúdo 3 </div>
