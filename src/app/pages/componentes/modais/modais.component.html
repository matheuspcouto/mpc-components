MPC Modais

Descrição: O componente mpc-modal é baseado na estrutura de modais do Bootstrap, que permite criar janelas modais responsivas e estilizadas. Os modais são utilizados para exibir informações, formulários ou mensagens ao usuário sem sair da página atual.
Propriedades:
  • titulo: O título exibido no cabeçalho do modal.
  • texto: O conteúdo principal exibido dentro do modal.
  • tipoModal: Define o tipo do modal, como confirmação, sucesso, aviso, etc.
  • botao: Função chamada ao clicar no botão principal do modal.
  • textoBotao: O texto exibido no botão principal.
  • segundoBotao: (opcional) Função chamada ao clicar no segundo botão, se houver.
  • textoSegundoBotao: (opcional) O texto exibido no segundo botão, se houver.
Importação no arquivo .ts:
imports: [ MpcModalComponent ]
Exemplo de uso no arquivo .ts:
@ViewChild('modalExemplo', { static: true }) modalExemplo!: MpcModalComponent;

abrirModalSucesso() {
  const modalSucesso: ModalConfig = {
    titulo: 'Modal de Sucesso',
    texto: 'Parabéns',
    tipoModal: TipoModal.SUCESSO,
    botao: () => { return true; },
    textoBotao: 'OK',
    segundoBotao: () => { this.modalExemplo?.fecharModal(); },
    textoSegundoBotao: 'Fechar',
  };
  this.modalExemplo?.abrirModal(modalSucesso);
}
Exemplo de uso no arquivo .html:
<mpc-modal #modalExemplo></mpc-modal>
