MPC Navbar

Descrição: O mpc-navbar é um componente de barra de navegação responsiva que suporta abas com sub-rotas e identificação automática de abas de login.
Propriedades:
  • abasInput: (obrigatório) Array de configuração das abas de navegação.
  • id: (opcional) O ID único do navbar para acessibilidade.
  • tabIndex: (opcional) O índice de tabulação para navegação por teclado.
  • ariaLabel: (opcional) Rótulo para leitores de tela.
Instalação:
npm install mpc-lib-angular
Importação no arquivo .ts:

import { MpcNavbarComponent, NavbarConfig, SubRotaConfig } from 'mpc-lib-angular';

// No seu componente
imports: [ MpcNavbarComponent ]
Exemplo de uso no arquivo .html:
<!-- Navbar Básico -->
<mpc-navbar [abasInput]="abas"></mpc-navbar>

<!-- Navbar com Acessibilidade -->
<mpc-navbar 
  [abasInput]="abas"
  id="navbar-principal"
  [tabIndex]="0"
  ariaLabel="Navegação principal">
</mpc-navbar>
Exemplo de configuração das abas no arquivo .ts:
abas: NavbarConfig[] = [
  { titulo: 'Home', rota: '/home', icone: 'bi bi-house-fill' },
  { titulo: 'Login', rota: '/login', icone: 'bi bi-person-fill' },
  {
    titulo: 'Componentes',
    rota: '/componentes',
    icone: 'bi bi-code-slash',
    subRotas: [
      { titulo: 'Buttons', rota: '/componentes/buttons' },
      { titulo: 'Cards', rota: '/componentes/cards' }
    ]
  }
];
Personalização com Variáveis CSS:
/* Cores da Navbar */
--mpc-color-bg-header: white;                    /* Cor de fundo da navbar */
--mpc-color-text-disabled: #7f7f90;             /* Cor do texto desabilitado */
--mpc-color-text-active: var(--mpc-color-primary); /* Cor do texto ativo */
--mpc-color-text-hover: var(--mpc-color-primary);  /* Cor do texto no hover */
--mpc-color-border-active: var(--mpc-color-secondary); /* Cor da borda ativa */

/* Fontes */
--mpc-font-navbar: var(--mpc-font-subtitle);    /* Fonte da navbar */
Interfaces:
  • NavbarConfig: Interface para configuração das abas
    • titulo: Título da aba (string)
    • rota: Rota de navegação (string)
    • icone: Classe do ícone Bootstrap (string)
    • subRotas: Array opcional de subrotas (SubRotaConfig[])
  • SubRotaConfig: Interface para configuração de sub-rotas
    • titulo: Título da subrota (string)
    • rota: Rota da subrota (string, opcional)
    • fragment: Fragmento da URL (string, opcional)
