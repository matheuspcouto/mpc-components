MPC Cards

Os mpc-card são baseados no framework Bootstrap, sendo componentes que permitem criar cards personalizados. Os cards são utilizados para exibir informações de forma organizada e atraente, com uma imagem, título, subtítulo e descrição. Abaixo, você pode ver como utilizar os componentes e seus parâmetros.

Cards Verticais e Horizontais

  • titulo: O título do card.
  • subtitulo: O subtítulo do card.
  • descricaoUm: A primeira descrição do card.
  • descricaoDois: A segunda descrição do card.
  • imagem: A imagem do card. Necessário ter a imagem na pasta "/assets/img". Exemplo: '/assets/img/imagem.jpg'.
  • links: (opcional) Os links do card. Em formato: [{ icone: 'string', url: 'string' }]
  • layout: (opcional) O layout do card, que pode ser: vertical ou horizontal. Padrão: vertical.
  • id: (opcional) O ID do card.
  • tabIndex: (opcional) O tabIndex do card.
  • ariaLabel: (opcional) O texto de acessibilidade do componente.
No arquivo seu-component.ts, importe o componente:
              
                ```
                imports: [ MpcCardComponent ]
                ```
No arquivo seu-component.html, adicione o seguinte código: 
                
                <mpc-card
          titulo="Card 1"
          subtitulo="Tamanho Vertical"
          descricaoUm="Primeira descrição do card"
          descricaoDois="Segunda descrição do card"
          imagem="/assets/img/no-image.jpg"
          [links]="[{icone: 'bi bi-linkedin', url: 'https://www.linkedin.com/in/matheuspcouto/'}]"
          layout="vertical">
        </mpc-card>

Cards com Imagem de Fundo

Os mpc-card-background-img são cards especialmente projetados para exibir informações com uma imagem de fundo, título, subtítulo e descrição.

Cards com Imagem de Fundo

O mpc-card-background-img é um card especial para exibir programação ou eventos com imagem de fundo, título, subtítulo e descrição.
  • titulo: O título do card.
  • subtitulo: O subtítulo do card (ex: horário).
  • descricao: A descrição do card.
  • imagemFundo: A imagem de fundo do card. Exemplo: '/assets/img/home/mountain.jpg'.
  • id: (opcional) O ID do card.
  • tabIndex: (opcional) O tabIndex do card.
  • ariaLabel: (opcional) O texto de acessibilidade do componente.
No arquivo seu-component.ts, importe o componente:
              
                ```
                imports: [ MpcCardBackGroundImgComponent ]
                ```
No arquivo seu-component.html, adicione o seguinte código: 
                
                <mpc-card-background-img
          titulo="Card 1"
          subtitulo="com imagem de fundo"
          descricao="Este é um card com imagem de fundo"
          imagemFundo="/assets/img/home/mountain.jpg"
          [tabIndex]="0"
          ariaLabel="Card 1">
        </mpc-card-background-img>
Para mais informações sobre como utilizar cards no Bootstrap, consulte a documentação oficial do Bootstrap sobre cards.

Cards de Menu

Os mpc-card-menu são cards simples para exibir informações rápidas de menu, como telefone, e-mail ou endereço, com ícone, título e descrição.

Cards de Menu

O mpc-card-menu é um card simples para exibir informações rápidas de menu, como telefone, e-mail ou endereço, com ícone, título e descrição.
  • titulo: O título do card.
  • descricao: (opcional) A descrição do card.
  • icone: O ícone do card (classe do Bootstrap Icons).
  • acao: (opcional) Função a ser executada ao clicar no card.
  • id: (opcional) O ID do card.
  • tabIndex: (opcional) O tabIndex do card.
  • ariaLabel: (opcional) O texto de acessibilidade do componente.
No arquivo seu-component.ts, importe o componente:
              
                ```ts
                imports: [ MpcCardMenuComponent ]
                ```
No arquivo seu-component.html, adicione o seguinte código: 
                
                <mpc-card-menu
          icone="bi bi-telephone"
          titulo="Telefone"
          descricao="(63) 9 9201-4337"
          (acao)="onMenuCardClick()">>
        </mpc-card-menu>
