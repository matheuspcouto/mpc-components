MPC Btn Float

Descrição: O <mpc-btn-float> do MpcComponents é um componente que exibe um botão flutuante fixo para rolar a página ao topo, permitindo personalização de ícone e cores.
Propriedades do Componente:
  • icone: string (opcional) — Classes CSS do ícone a ser exibido (ex: 'bi bi-arrow-up-short').
  • disabled: boolean (opcional, padrão: false) — Indica se o botão está desabilitado.
  • id: string (opcional) — ID do botão para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
Importação no arquivo .ts:

import { MpcBtnFloatComponent } from 'mpc-lib-angular';

// No seu componente
imports: [MpcBtnFloatComponent]

Observações importantes:
  • O componente <mpc-btn-float> já aplica automaticamente os estilos CSS necessários ao elemento.
  • O ícone é inserido dinamicamente pelo Angular.
  • As cores podem ser definidas em qualquer formato CSS válido (hex, rgb, rgba, nomes de cores, variáveis CSS).
  • Para eventos de clique, utilize (click)="suaFuncao()".
  • O componente inclui suporte para acessibilidade.
