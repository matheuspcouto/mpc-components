MPC Inputs

Descrição: Os mpc-input-* são componentes customizados baseados no Bootstrap para entrada de dados, como texto, número, data, e-mail, senha, telefone, CPF/CNPJ, CEP, área de texto, rádio e select. Eles oferecem validação, acessibilidade e integração fácil com formulários Angular.
Propriedades comuns:
  • label: Label do campo (quando aplicável).
  • value: Valor do campo.
  • id: (opcional) Id do campo.
  • tabIndex: (opcional) Índice de tabulação do campo.
  • ariaLabel: (opcional) Label para acessibilidade.
  • readonly: (opcional) Campo somente leitura.
  • disabled: (opcional) Indica se o campo está desabilitado.
  • required: (opcional) Indica se o campo é obrigatório.
  • min: (opcional) Valor mínimo ou mínimo de caracteres (quando aplicável).
  • max: (opcional) Valor máximo ou máximo de caracteres (quando aplicável).
  • options: (apenas para rádio e select) Lista de opções do campo.

Inicialização do formulário no arquivo .ts:
private readonly formBuilder = inject(NonNullableFormBuilder);

protected form = this.formBuilder.group({
  nome: [''],
  idade: [0],
  dataNasc: [''],
  email: [''],
  senha: [''],
  telefone: [''],
  cpfCnpj: [''],
  descricao: [''],
  cep: [''],
  rua: [''],
  numero: [''],
  complemento: [''],
  bairro: [''],
  cidade: [''],
  estado: [''],
});
Exemplo de uso no arquivo .html:
<mpc-input-text label="Nome" [value]="form.value.nome" (valor)="form.controls.nome.setValue($event)" (error)="form.controls.nome.setErrors($event)" min="3" />

<mpc-input-number label="Idade" [value]="form.value.idade" (valor)="form.controls.idade.setValue($event)" (error)="form.controls.idade.setErrors($event)" [min]="18" [max]="100" />

<mpc-input-date label="Data de Nascimento" [value]="form.value.dataNasc" (valor)="form.controls.dataNasc.setValue($event)" (error)="form.controls.dataNasc.setErrors($event)" [minDate]="minDate" [required]="true" [maxDate]="maxDate" />

<mpc-input-email [value]="form.value.email" (valor)="form.controls.email.setValue($event)" (error)="form.controls.email.setErrors($event)" [required]="true" />

<mpc-input-senha [value]="form.value.senha" (valor)="form.controls.senha.setValue($event)" (error)="form.controls.senha.setErrors($event)" [required]="true" />

<mpc-input-telefone [value]="form.value.telefone" (valor)="form.controls.telefone.setValue($event)" (error)="form.controls.telefone.setErrors($event)" [required]="true" />

<mpc-input-cpfcnpj [value]="form.value.cpfCnpj" (valor)="form.controls.cpfCnpj.setValue($event)" (error)="form.controls.cpfCnpj.setErrors($event)" [required]="true" />

<mpc-input-busca-cep [value]="form.value.cep" [required]="true" (valor)="definirEnderecoPorCep($event)" (error)="form.controls.cep.setErrors($event)" />

<mpc-input-text-area label="Descrição" [value]="form.value.descricao" (valor)="form.controls.descricao.setValue($event)" (error)="form.controls.descricao.setErrors($event)" min="10" max="100" />

<mpc-input-radio label="Sexo" [options]="sexos" [required]="true" />

<mpc-input-select label="Estado" [options]="estadosCivis" [required]="true" />
