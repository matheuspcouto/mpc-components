Descrição: O <mpc-button> do MpcComponents é um componente estilizado, permitindo a inclusão de texto e/ou ícones, personalização de cores e acessibilidade.
Propriedades do Componente:
  • texto: string (opcional) — Texto exibido no botão.
  • icone: string (opcional) — Classes do ícone (ex: 'bi bi-check').
  • posicaoIcone: 'esquerda' | 'direita' (opcional, padrão: 'direita') — Posição do ícone em relação ao texto.
  • disabled: boolean (opcional, padrão: false) — Indica se o botão está desabilitado.
  • id: string (opcional) — ID do botão para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
Variáveis CSS Personalizáveis:
  • --mpc-bg-color-btn: Cor de fundo padrão do botão (padrão: var(--mpc-color-primary)).
  • --mpc-text-color-btn: Cor do texto padrão do botão (padrão: white).
  • --mpc-bg-color-btn-hover: Cor de fundo do botão ao passar o mouse (padrão: mesma cor do fundo).
  • --mpc-text-color-btn-hover: Cor do texto do botão ao passar o mouse (padrão: mesma cor do texto).
Importação no arquivo .ts:

import { MpcButtonComponent } from 'mpc-lib-angular';

// No seu componente
imports: [MpcButtonComponent]

Observações importantes:
  • O componente <mpc-button> aplica automaticamente a classe CSS .mpc-btn.
  • O botão tem largura mínima de 30vh e máxima de 50vh (30vh em telas grandes).
  • O botão tem bordas arredondadas (border-radius: 9999px) e padding interno.
  • A posição padrão do ícone é direita.
  • As cores são personalizáveis através das variáveis CSS do componente.
  • Para eventos de clique, utilize (click)="suaFuncao()".
  • O componente inclui suporte completo para acessibilidade.
  • Botões desabilitados têm opacidade reduzida e cursor não permitido.
