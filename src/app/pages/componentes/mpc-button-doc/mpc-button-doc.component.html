MPC Button

Descrição: O <mpc-button> do MpcComponents é um componente estilizado, permitindo a inclusão de ícones e personalização via propriedades.
Propriedades do Componente:
  • texto: string (opcional) — Texto exibido no botão.
  • icone: string (opcional) — Classes do ícone (ex: 'bi bi-check').
  • posicaoIcone: 'esquerda' | 'direita' (opcional, padrão: 'direita') — Posição do ícone em relação ao texto.
  • disabled: boolean (opcional, padrão: false) — Indica se o botão está desabilitado.
  • id: string (opcional) — ID do botão para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
Propriedades CSS do Elemento:
  • [style.background-color]: Cor personalizada para o fundo do botão (hex, rgb, rgba, nome da cor).
  • [style.color]: Cor personalizada para o texto do botão (hex, rgb, rgba, nome da cor).
  • disabled: (opcional) Atributo que indica se o botão está desabilitado.
  • tabindex: (opcional) Índice de tabulação para acessibilidade.
  • aria-label: (opcional) Descrição do botão para acessibilidade.
Importação no arquivo .ts:

Observações importantes:
  • O componente <mpc-button> já aplica a classe CSS .mpc-btn automaticamente.
  • O texto e ícone são inseridos dinamicamente pelo Angular.
  • As cores são definidas através das propriedades CSS [style.background-color] e [style.color].
  • A posição padrão do ícone é direita.
  • As cores podem ser definidas em qualquer formato CSS válido (hex, rgb, rgba, nomes de cores, variáveis CSS).
  • Para eventos de clique, utilize (click)="suaFuncao()".
  • O componente inclui suporte para acessibilidade com tabindex e aria-label.
