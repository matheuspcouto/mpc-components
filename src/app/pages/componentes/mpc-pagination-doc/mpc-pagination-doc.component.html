Descrição: O mpc-pagination é um componente de paginação customizado que exibe uma barra de navegação de páginas com suporte a seleção de itens por página, navegação e acessibilidade.
Propriedades do Componente:
  • totalItens: number (obrigatório) — Número total de itens a serem paginados.
  • mostrarSeletorItensPorPagina: boolean (opcional, padrão: true) — Mostra seletor de itens por página.
  • id: string (opcional) — ID do componente para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação do componente.
  • ariaLabel: string (opcional) — Label para acessibilidade.
Instalação:
npm install mpc-lib-angular
Importação no arquivo .ts:

import { MpcPaginationComponent } from 'mpc-lib-angular';

// No seu componente
imports: [ MpcPaginationComponent ]
Exemplo de uso no arquivo .html:
<!-- Paginação Básica -->
<mpc-pagination 
  [totalItens]="itensTotais.length" 
  (indices)="definirIndiceLista($event)" 
  [mostrarSeletorItensPorPagina]="true">
</mpc-pagination>

<!-- Paginação com Acessibilidade -->
<mpc-pagination 
  [totalItens]="itensTotais.length" 
  (indices)="definirIndiceLista($event)" 
  [mostrarSeletorItensPorPagina]="true"
  id="pagination-exemplo"
  [tabIndex]="0"
  ariaLabel="Navegação de páginas">
</mpc-pagination>
Personalização com Variáveis CSS:
/* Cores da Paginação */
--mpc-color-text-pagination: var(--mpc-color-primary); /* Cor do texto da paginação */
--mpc-color-bg-pagination: white;                      /* Cor de fundo da paginação */
--mpc-color-border-pagination: var(--mpc-color-primary); /* Cor da borda da paginação */
--mpc-color-active-pagination: var(--mpc-color-primary); /* Cor da página ativa */
--mpc-color-hover-pagination: var(--mpc-color-secondary); /* Cor do hover */

/* Fontes */
--mpc-font-pagination: var(--mpc-font-default);        /* Fonte da paginação */
Eventos:
  • indices: Emite um objeto { indiceInicial: number, indiceFinal: number } indicando o range de itens exibidos na página atual.
