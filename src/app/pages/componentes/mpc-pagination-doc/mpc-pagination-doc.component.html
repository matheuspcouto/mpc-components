MPC Pagination

Descrição: O mpc-pagination é um componente de paginação customizado que exibe uma barra de navegação de páginas com suporte a seleção de itens por página, navegação e acessibilidade.
Propriedades:
  • @Input id (string, opcional): Id do componente para acessibilidade.
  • @Input tabIndex (number, opcional): Índice de tabulação do componente.
  • @Input ariaLabel (string, opcional): Label para acessibilidade.
  • @Input totalItens (number, obrigatório): Número total de itens a serem paginados.
  • @Input mostrarSeletorItensPorPagina (boolean, opcional): Indica se deve mostrar o seletor de itens por página (padrão: true).
  • @Input textColor (string, opcional): Cor do texto dos elementos da paginação.
  • @Output indices: Emite um objeto {{ '{' }} indiceInicial, indiceFinal {{ '}' }} indicando o range de itens exibidos na página atual.
Importação no arquivo .ts:
imports: [ MpcPaginationComponent ]
Exemplo de uso no arquivo .html:
<mpc-pagination textColor="var(--mpc-color-primary)" [totalItens]="itensTotais.length" (indices)="definirIndiceLista($event)" [mostrarSeletorItensPorPagina]="true"></mpc-pagination>
