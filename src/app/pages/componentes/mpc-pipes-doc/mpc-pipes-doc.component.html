MPC Pipes

Descrição: Os pipes de máscara da MPC facilitam a formatação de valores como CEP, CPF/CNPJ, telefone e e-mail diretamente no template.
Instalação:
npm install mpc-lib-angular
Importação no arquivo .ts:
import { CepMaskPipe, CpfCnpjMaskPipe, TelefoneMaskPipe, EmailMaskPipe } from 'mpc-lib-angular';
// No seu componente
imports: [CepMaskPipe, CpfCnpjMaskPipe, TelefoneMaskPipe, EmailMaskPipe]
Exemplo de Uso dos Pipes:
<p>{{ ' 12345678901' + ' | ' + 'cpfCnpjMaskPipe ' }}</p>
          <p>{{ ' 12345678' + ' | ' + 'cepMaskPipe ' }}</p>
          <p>{{ ' 11999999999' + ' | ' + 'telefoneMaskPipe ' }}</p>
Observação: Para utilizar os pipes standalone, é necessário importá-los no array imports do seu componente Angular, conforme o exemplo acima.

Pipes de Máscara

CEP

Entrada: 12345678

Saída: {{ '12345678' | cepMaskPipe }}

CPF

Entrada: 12345678901

Saída: {{ '12345678901' | cpfCnpjMaskPipe }}

CNPJ

Entrada: 12345678000199

Saída: {{ '12345678000199' | cpfCnpjMaskPipe }}

Telefone

Entrada: 11987654321

Saída: {{ '11987654321' | telefoneMaskPipe }}

E-mail

Entrada: usuario@exemplo.com

Saída: {{ 'usuario@exemplo.com' | emailMaskPipe }}
