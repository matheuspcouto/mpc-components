MPC Page Header

Descrição: O mpc-page-header é um componente de cabeçalho de página, ideal para destacar o título e subtítulo de cada tela.
Propriedades:
  • titulo: O título do cabeçalho.
  • subtitulo: (opcional) O subtítulo do cabeçalho.
Importação no arquivo .ts:
imports: [ MpcPageHeaderComponent ]
Exemplo de uso no arquivo .html:
<mpc-page-header titulo="Título da Página" subtitulo="Subtítulo opcional"></mpc-page-header>
