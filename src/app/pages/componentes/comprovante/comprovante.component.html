MPC Comprovantes

O componente mpc-comprovante é baseado na estrutura de modais do Bootstrap, que permite criar janelas modais responsivas e estilizadas. Os modais são utilizados para exibir comprovantes de inscrição, pagamento e outras informações importantes ao usuário sem sair da página atual.
No arquivo seu-component.ts, importe o componente:
            
import { MpcComprovanteComponent, MpcComprovanteConfig } from './caminho/para/mpc-comprovante.component';

@Component({
  selector: 'seu-component',
  imports: [MpcComprovanteComponent],
  templateUrl: './seu-component.html'
})
Ainda no seu-component.ts, crie uma nova instância do comprovante e defina um método de abertura do comprovante com seus parâmetros:
            
@ViewChild('comprovanteExemplo', { static: true }) comprovanteExemplo!: MpcComprovanteComponent;
dadosComprovante: MpcComprovanteConfig = {} as MpcComprovanteConfig;

abrirModalComprovante() {
  this.dadosComprovante = {
    titulo: 'Comprovante de Inscrição',
    dados: {
      dadosInscricao: {
        codigoInscricao: '123456789',
        dataInscricao: '2025-02-27',
        status: 'ATIVO'
      },
      dadosPessoais: [
        { label: 'Nome Completo', valor: 'João Silva Santos' },
        { label: 'CPF', valor: '123.456.789-00' },
        { label: 'E-mail', valor: 'joao.silva@email.com' },
        { label: 'Telefone', valor: '(11) 99999-9999' }
      ],
      dadosPagamento: {
        formaPagamento: 'PIX',
        valor: 150.00,
        statusPagamento: 'A PAGAR',
        dataPagamento: '2025-02-28'
      }
    }
  };

  this.comprovanteExemplo?.abrirComprovante();
}
Interface MpcComprovanteConfig:
            
export interface MpcComprovanteConfig {
  titulo: string;
  dados: {
    dadosInscricao: {
      codigoInscricao: string;
      dataInscricao: string;
      status?: string; // 'ATIVO' | 'INATIVO'
    }
    dadosPessoais: { label: string; valor: string }[];
    dadosPagamento: {
      formaPagamento: string;
      valor: number;
      statusPagamento?: string; // 'PAGO' | 'A PAGAR'
      dataPagamento?: string;
    }
  }
}
No arquivo seu-component.html, adicione o seguinte código: 
            
<mpc-comprovante
  [comprovante]="dadosComprovante"
  #comprovanteExemplo>
</mpc-comprovante>
Funcionalidades do Componente:
  • Exibição de dados de inscrição: Código, data e status da inscrição
  • Dados pessoais dinâmicos: Lista configurável de informações pessoais
  • Informações de pagamento: Forma de pagamento, valor e status
  • Cópia de código: Botão para copiar código de inscrição
  • Status visuais: Badges coloridos para status de inscrição e pagamento
  • Formatação automática: Valores monetários e datas formatados
  • Responsivo: Adaptável a diferentes tamanhos de tela
Clique no botão abaixo e veja como ficou a implementação.
Métodos Disponíveis:
  • abrirComprovante() - Abre o modal do comprovante
  • fecharComprovante() - Fecha o modal do comprovante
  • copiarCodigo(valor: string) - Copia o código para a área de transferência
  • formatarValor(valor: number) - Formata valores monetários em R$
  • formatarData(data: string) - Formata datas no padrão brasileiro
Status Suportados:
Status de Inscrição:
  • ATIVO - Inscrição ativa
  • INATIVO - Inscrição inativa
Status de Pagamento:
  • PAGO - Pagamento realizado
  • A PAGAR - Aguardando pagamento
Para mais informações sobre como utilizar modais no Bootstrap, consulte a documentação oficial do Bootstrap sobre modais.
