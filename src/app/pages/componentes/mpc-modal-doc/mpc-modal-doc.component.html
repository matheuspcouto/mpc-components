MPC Modais

Descrição: O componente mpc-modal exibe modais personalizados com diferentes tipos de configuração, suportando títulos, textos, imagens e botões customizáveis. O componente é controlado via métodos e não possui }Input() - é gerenciado programaticamente.
Propriedades do Componente:
  • id: string (opcional) — ID do modal para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
O componente não possui outros @Input(). O controle de exibição e conteúdo é feito via métodos públicos e configuração (MpcModalConfig).
Importação no arquivo .ts:

import { MpcModalComponent, MpcModalConfig, TipoModal } from 'mpc-lib-angular';

// No seu componente
imports: [MpcModalComponent]

Exemplo de uso no arquivo .ts:

@ViewChild('modalExemplo', { static: true }) modalExemplo!: MpcModalComponent;

abrirModalSucesso() {
  const modalSucesso: MpcModalConfig = {
    titulo: 'Modal de Sucesso',
    texto: 'Operação realizada com sucesso!',
    tipoModal: TipoModal.SUCESSO,
    botao: () => { return true; },
    textoBotao: 'OK',
    segundoBotao: () => { this.modalExemplo?.fecharModal(); },
    textoSegundoBotao: 'Fechar',
  };
  this.modalExemplo?.abrirModal(modalSucesso);
}
Personalização com Variáveis CSS:
/* Cores do Modal */
--mpc-color-bg-modal: white;                    /* Cor de fundo */
--mpc-color-title-modal: var(--mpc-color-primary); /* Cor do título */
--mpc-color-description-modal: var(--mpc-color-tertiary); /* Cor da descrição */

/* Cores dos Botões */
--mpc-color-first-button-bg-modal: var(--mpc-color-primary); /* Fundo do primeiro botão */
--mpc-color-first-button-text-modal: white;     /* Texto do primeiro botão */
--mpc-color-second-button-bg-modal: var(--mpc-color-secondary); /* Fundo do segundo botão */
--mpc-color-second-button-text-modal: white;    /* Texto do segundo botão */
--mpc-color-copy-button-bg-modal: transparent;  /* Fundo do botão de copiar */
--mpc-color-copy-button-text-modal: var(--mpc-color-primary); /* Texto do botão de copiar */

/* Fontes */
--mpc-font-title-modal: var(--mpc-font-subtitle); /* Fonte do título */
--mpc-font-description-modal: var(--mpc-font-default); /* Fonte da descrição */
--mpc-font-button-modal: var(--mpc-font-default); /* Fonte dos botões */
Veja e teste abaixo os tipos de modais disponíveis
Tipos de Modal Disponíveis:
  • CONFIRMACAO: Modal de confirmação (metade da tela)
  • ALERTA: Modal de alerta (tela inteira, imagem de atenção)
  • ERRO: Modal de erro (tela inteira, imagem de erro)
  • SUCESSO: Modal de sucesso (tela inteira, imagem de sucesso)
