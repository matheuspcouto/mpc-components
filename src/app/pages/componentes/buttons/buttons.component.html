MPC Buttons

O mpc-button do MpcComponents são baseados no framework Bootstrap, que oferece uma ampla gama de estilos e funcionalidades para botões. Abaixo, você pode ver como utilizar o componente e seus parâmetros.
  • id: (opcional) Um identificador único para o botão.
  • texto: O texto que será exibido no botão.
  • corFundo: Cor personalizada para o fundo do botão (hex, rgb, rgba, nome da cor).
  • corTexto: Cor personalizada para o texto do botão (hex, rgb, rgba, nome da cor).
  • disabled: (opcional) Um booleano que indica se o botão deve estar desabilitado.
  • posicaoIcone: (opcional) Define a posição do ícone em relação ao texto, podendo ser esquerda ou direita. Padrão: direita.
  • icone: (opcional) O ícone a ser exibido no botão, utilizando classes do Bootstrap Icons ou a que preferir.
  • tabIndex: (opcional) O índice de tabulação para acessibilidade.
  • ariaLabel: (opcional) Uma descrição do botão para acessibilidade.
No arquivo seu-component.ts, importe o componente:
              
                ```
                imports: [ MpcButtonComponent ]
                ```
No arquivo seu-component.html, adicione o seguinte código:
              
                <mpc-button (click)="minhaFuncao()"
                [corFundo]="'#17a2b8'"
                [corTexto]="'white'"
                [icone]="'bi bi-code-slash'"
                [posicaoIcone]="'esquerda'"
                [texto]="'Exemplo de Botão'">
              </mpc-button>

Observações importantes:
  • O componente utiliza cores personalizadas através das propriedades corFundo e corTexto.
  • A posição padrão do ícone é direita.
  • As cores podem ser definidas em qualquer formato CSS válido (hex, rgb, rgba, nomes de cores).
  • Para eventos de clique, utilize (click)="suaFuncao()".
  • O componente inclui suporte completo para acessibilidade com tabIndex e ariaLabel.
