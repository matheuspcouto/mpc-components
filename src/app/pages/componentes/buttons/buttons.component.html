MPC Buttons

Descrição: O mpc-button do MpcComponents é baseado no framework Bootstrap, que oferece uma ampla gama de estilos e funcionalidades para botões.
Propriedades:
  • id: (opcional) Identificador único para o botão.
  • texto: O texto exibido no botão.
  • corFundo: Cor personalizada para o fundo do botão (hex, rgb, rgba, nome da cor).
  • corTexto: Cor personalizada para o texto do botão (hex, rgb, rgba, nome da cor).
  • disabled: (opcional) Booleano que indica se o botão está desabilitado.
  • posicaoIcone: (opcional) Define a posição do ícone em relação ao texto: esquerda ou direita. Padrão: direita.
  • icone: (opcional) Ícone exibido no botão, utilizando classes do Bootstrap Icons ou outra de sua preferência.
  • tabIndex: (opcional) Índice de tabulação para acessibilidade.
  • ariaLabel: (opcional) Descrição do botão para acessibilidade.
Importação no arquivo .ts:
imports: [ MpcButtonComponent ]
Exemplo de uso no arquivo .html:
<mpc-button (click)="minhaFuncao()" [corFundo]="'#17a2b8'" [corTexto]="'white'" [icone]="'bi bi-code-slash'" [posicaoIcone]="'esquerda'" [texto]="'Exemplo de Botão'"></mpc-button>
Observações importantes:
  • O componente utiliza cores personalizadas através das propriedades corFundo e corTexto.
  • A posição padrão do ícone é direita.
  • As cores podem ser definidas em qualquer formato CSS válido (hex, rgb, rgba, nomes de cores).
  • Para eventos de clique, utilize (click)="suaFuncao()".
  • O componente inclui suporte completo para acessibilidade com tabIndex e ariaLabel.
