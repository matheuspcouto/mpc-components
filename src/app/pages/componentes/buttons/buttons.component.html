MPC Buttons

O mpc-button do MyComponents são baseados no framework Bootstrap, que oferece uma ampla gama de estilos e funcionalidades para botões. Abaixo, você pode ver como utilizar o componente e seus parâmetros.
  • id: (opcional) Um identificador único para o botão.
  • texto: O texto que será exibido no botão.
  • corBotao: A classe de cor do botão, que pode ser qualquer uma das classes de botão do Bootstrap. Padrão: btn-primary
  • disabled: (opcional) Um booleano que indica se o botão deve estar desabilitado.
  • posicaoIcone: (opcional) Define a posição do ícone em relação ao texto, podendo ser esquerda ou direita. Padrão: esquerda.
  • icone: (opcional) O ícone a ser exibido no botão, utilizando classes do Bootstrap Icons ou a que preferir.
  • tabIndex: (opcional) O índice de tabulação para acessibilidade.
  • ariaLabel: (opcional) Uma descrição do botão para acessibilidade.
No seu arquivo component.ts, importe o componente:
              
                ```
                imports: [ MpcButtonComponent ]
                ```
No seu arquivo component.html, adicione o seguinte código:
              
                <mpc-button (click)="" [corBotao]="'btn-info'"
                [icone]="'bi bi-code-slash'" [posicaoIcone]="'esquerda'" [texto]="'Exemplo de Botão'>
              </mpc-button>
