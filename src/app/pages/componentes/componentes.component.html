MPC Loader

O mpc-loader é um componente que exibe uma animação de carregamento enquanto o conteúdo da página está sendo carregado a cada requisição HTTP. Pode ser colocado em qualquer lugar da sua página, mas caso deseje utilizar para todas as páginas do projeto, incluá-o no arquivo app.component.html.
Veja como é a implementação deste componente no arquivo .html no exemplo abaixo. 
              
              <mpc-loader></mpc-loader>

MPC Tabs

O mpc-tabs é um componente que permite criar tabs personalizados. Os tabs são utilizados para exibir informações de forma organizada e atraente.
Veja como é a implementação deste componente no arquivo .html no exemplo abaixo. 
              
                <mpc-tabs [tabs]="tabs" [tabIndex]="0" ariaLabel="Tabs" (tabSelected)="selectTab($event)"></mpc-tabs>
Os parâmetros deste componente são os seguintes:
  • tabs: Os tabs do componente. Formato: [{ id: 'string', titulo: 'string' }]
  • tabIndex: O tabIndex do card.
  • ariaLabel: O texto de acessibilidade do componente.

MPC Modais

O componente mpc-modal é baseado na estrutura de modais do Bootstrap, que permite criar janelas modais responsivas e estilizadas. Os modais são utilizados para exibir informações, formulários ou mensagens ao usuário sem sair da página atual.
Veja como é a implementação deste componente no arquivo .html no exemplo abaixo. 
            
            <mpc-modal #modalExemplo></mpc-modal>
Os parâmetros deste componente são enviados no arquivo .ts com os seguintes campos:
  • titulo: O título que será exibido no cabeçalho do modal.
  • texto: O conteúdo principal que será exibido dentro do modal.
  • tipoModal: Define o tipo do modal, como confirmação, sucesso, aviso, etc.
  • botao: A função que será chamada ao clicar no botão principal do modal.
  • textoBotao: O texto que será exibido no botão principal.
  • segundoBotao: (opcional) A função que será chamada ao clicar no segundo botão, se houver.
  • textoSegundoBotao: (opcional) O texto que será exibido no segundo botão, se houver.
          
            @ViewChild('modalExemplo', { static: true }) modalExemplo!: MpcModalComponent;
          
        
              
                abrirModalSucesso() {
                  const modalSucesso: ModalConfig = {
                    titulo: 'Modal de Sucesso',
                    texto: 'Parabéns',
                    tipoModal: TipoModal.SUCESSO,
                    botao: () => { return true; },
                    textoBotao: 'OK',
                    segundoBotao: () => { this.modalExemplo?.fecharModal(); },
                    textoSegundoBotao: 'Fechar',
                  }

                  this.modalExemplo.modal = modalSucesso;
                  this.modalExemplo?.abrirModal();
                }
              
            
              
                abrirModalConfirmacao() {
                  const modalConfirmacao: ModalConfig = {
                  titulo: 'Modal de Confirmação',
                  texto: 'Este é um modal de confirmação',
                  tipoModal: TipoModal.CONFIRMACAO,
                  botao: () => { return true; },
                  textoBotao: 'OK',
                  segundoBotao: () => { this.modalExemplo?.fecharModal(); },
                  textoSegundoBotao: 'Fechar',
                };

                this.modalExemplo.modal = modalConfirmacao;
                this.modalExemplo?.abrirModal();
                }
              
            
              
                abrirModalAlerta() {
                  const modalAlerta: ModalConfig = {
                    titulo: 'Modal de Alerta',
                    texto: 'Este é um modal de alerta',
                    tipoModal: TipoModal.ALERTA,
                    botao: () => { return true; },
                    textoBotao: 'OK',
                    segundoBotao: () => { this.modalExemplo?.fecharModal(); },
                    textoSegundoBotao: 'Fechar',
                  }

                  this.modalExemplo.modal = modalAlerta;
                  this.modalExemplo?.abrirModal();
                }
              
            
              
                abrirModalErro() {
                  const modalErro: ModalConfig = {
                    titulo: 'Modal de Erro',
                    texto: 'Este é um modal de erro',
                    tipoModal: TipoModal.ERRO,
                    botao: () => this.modalExemplo?.fecharModal(),
                    textoBotao: 'OK',
                    // Não é necessário passar o segundo botão
                    // O modal de erro já possui um botão de copiar o erro
                  }

                  this.modalExemplo.modal = modalErro;
                  this.modalExemplo?.abrirModal();
                }
Para mais informações sobre como utilizar modais no Bootstrap, consulte a documentação oficial do Bootstrap sobre modais.
