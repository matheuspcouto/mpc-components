MPC Buttons

Os botões do MyComponents são baseados no framework Bootstrap, que oferece uma ampla gama de estilos e funcionalidades para botões. Abaixo, você pode ver como utilizar o componente mpc-button e seus parâmetros.
Os parâmetros que você pode passar para o componente mpc-button incluem:
  • id: (opcional) Um identificador único para o botão.
  • texto: O texto que será exibido no botão.
  • corBotao: A classe de cor do botão, que pode ser qualquer uma das classes de botão do Bootstrap.
  • tabIndex: O índice de tabulação para acessibilidade.
  • disabled: (opcional) Um booleano que indica se o botão deve estar desabilitado.
  • posicaoIcone: (opcional) Define a posição do ícone em relação ao texto, podendo ser esquerda ou direita. Padrão: esquerda.
  • icone: O ícone a ser exibido no botão, utilizando classes do Bootstrap Icons ou a que preferir.
Para mais informações sobre como utilizar botões no Bootstrap, consulte a documentação oficial do Bootstrap sobre botões. 
              
                <mpc-button [disabled]="false" (click)="" [corBotao]="'btn-info'"
                [icone]="'bi bi-code-slash'" [posicaoIcone]="'esquerda'" [tabIndex]="0" [texto]="'Exemplo de Botão'>
              </mpc-button>

MPC Modais

O componente mpc-modal é baseado na estrutura de modais do Bootstrap, que permite criar janelas modais responsivas e estilizadas. Os modais são utilizados para exibir informações, formulários ou mensagens ao usuário sem sair da página atual.
Veja como é a implementação deste componente no arquivo .html no exemplo abaixo. 
            
            <mpc-modal #modalExemplo></mpc-modal>
Os parâmetros deste componente são enviados no arquivo .ts com os seguintes campos:
  • titulo: O título que será exibido no cabeçalho do modal.
  • texto: O conteúdo principal que será exibido dentro do modal.
  • tipoModal: Define o tipo do modal, como confirmação, sucesso, aviso, etc.
  • botao: A função que será chamada ao clicar no botão principal do modal.
  • textoBotao: O texto que será exibido no botão principal.
  • segundoBotao: (opcional) A função que será chamada ao clicar no segundo botão, se houver.
  • textoSegundoBotao: (opcional) O texto que será exibido no segundo botão, se houver.
              
                @ViewChild('modalExemplo', { static: true }) modalExemplo!: MpcModalComponent;
              
            
              
                abrirModalSucesso() {
                  const modalSucesso: ModalConfig = {
                    titulo: 'Modal de Sucesso',
                    texto: 'Parabéns',
                    tipoModal: TipoModal.SUCESSO,
                    botao: () => { return true; },
                    textoBotao: 'OK',
                    segundoBotao: () => { this.modalExemplo?.fecharModal(); },
                    textoSegundoBotao: 'Fechar',
                  }

                  this.modalExemplo.modal = modalSucesso;
                  this.modalExemplo?.abrirModal();
                }

                    texto: 'Este é um modal de alerta',
                    tipoModal: TipoModal.ALERTA,
                    botao: () => { return true; },
                    textoBotao: 'OK',

                    textoSegundoBotao: 'Fechar',

                  this.modalExemplo?.abrirModal();
                }
              
            
              
                @ViewChild('modalExemplo', { static: true }) modalExemplo!: MpcModalComponent;
              
            
              
                abrirModalErro() {
                  const modalErro: ModalConfig = {
                    titulo: 'Modal de Erro',
                    texto: 'Este é um modal de erro',
                    tipoModal: TipoModal.ERRO,
                    botao: () => this.modalExemplo?.fecharModal(),
                    textoBotao: 'OK',
                    // Não é necessário passar o segundo botão
                    // O modal de erro já possui um botão de copiar o erro
                  }

                  this.modalExemplo.modal = modalErro;
                  this.modalExemplo?.abrirModal();
                }
Para mais informações sobre como utilizar modais no Bootstrap, consulte a documentação oficial do Bootstrap sobre modais.

Notificações & Loader

As notificações do MyComponents são geradas pelo componente "NotificationService", da biblioteca "ngx-toastr".
O Loader é gerado pelo componente "LoaderService", localizado no diretório "src/app/shared/components/loader".

MPC Comprovantes

O componente mpc-comprovante é baseado na estrutura de modais do Bootstrap, que permite criar janelas modais responsivas e estilizadas. Os modais são utilizados para exibir informações, formulários ou mensagens ao usuário sem sair da página atual.
Veja como é a implementação deste componente no arquivo .html no exemplo abaixo. 
            
            <mpc-comprovante [comprovante]="dadosComprovante" #comprovanteExemplo></mpc-comprovante>
Agora, veja como é a implementação deste componente no arquivo .ts
              
                @ViewChild('comprovanteExemplo', { static: true }) comprovanteExemplo!: MpcComprovanteComponent;
                dadosComprovante: MpcComprovanteConfig = {} as MpcComprovanteConfig;
              
            
              
                abrirModalComprovante() {
                  this.dadosComprovante = {
                    titulo: 'Comprovante de inscrição',
                    dados: {
                      dadosInscricao: {
                        codigoInscricao: '123456',
                        dataInscricao: new Date().toLocaleDateString('pt-BR'),
                        status: 'ATIVO',
                      },
                      dadosPessoais: [
                        { label: 'Nome', valor: 'Fulano de Tal' },
                        { label: 'CPF', valor: '123.456.789-00' },
                        { label: 'E-mail', valor: 'fulano@example.com' },
                      ],
                      dadosPagamento: {
                        formaPagamento: 'Cartão de Crédito',
                        valor: 100.00,
                        statusPagamento: 'A PAGAR'
                      },
                    }
                  };

                  this.comprovanteExemplo?.abrirComprovante();
                }
