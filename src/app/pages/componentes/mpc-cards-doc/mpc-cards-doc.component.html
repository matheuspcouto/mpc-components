MPC Cards

Descrição: O mpc-card é um componente para exibir cards com layout vertical e horizontal, suportando imagens, títulos, subtítulos, descrições e links com design customizável.
Propriedades:
  • titulo: string (obrigatório) — O título principal do card.
  • subtitulo: string (opcional) — O subtítulo do card.
  • descricaoUm: string (opcional) — Primeira descrição do card.
  • descricaoDois: string (opcional) — Segunda descrição do card.
  • imagem: string (opcional) — URL da imagem do card.
  • links: Array<{ url: string; icone: string }> (opcional) — Array de links com ícones.
  • layout: 'vertical' | 'horizontal' (opcional, padrão: 'vertical') — Layout do card.
  • id: string (opcional) — ID único do card para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
Instalação:
npm install mpc-lib-angular
Importação no arquivo .ts:

import { MpcCardComponent } from 'mpc-lib-angular';

// No seu componente
imports: [MpcCardComponent]
Exemplo de uso no arquivo .html:
<!-- Card Vertical Básico -->
<mpc-card
  titulo="Título do Card"
  subtitulo="Subtítulo"
  descricaoUm="Primeira descrição do card"
  descricaoDois="Segunda descrição do card"
  imagem="/assets/img/exemplo.jpg"
  [links]="[{icone: 'bi bi-linkedin', url: 'https://linkedin.com'}]"
  [layout]="'vertical'">
</mpc-card>

<!-- Card Horizontal com Links -->
<mpc-card
  titulo="Card Horizontal"
  subtitulo="Com Links"
  descricaoUm="Descrição do card horizontal"
  imagem="/assets/img/exemplo.jpg"
  [layout]="'horizontal'"
  [links]="[
    {icone: 'bi bi-linkedin', url: 'https://linkedin.com'},
    {icone: 'bi bi-github', url: 'https://github.com'}
  ]">
</mpc-card>

Cards com Background

Descrição: O mpc-card-background-img é um componente para exibir cards com imagem de fundo e gradiente escuro, ideal para destacar informações importantes com visual impactante.
Propriedades:
  • titulo: string (obrigatório) — O título principal do card.
  • subtitulo: string (opcional) — O subtítulo do card.
  • descricao: string (opcional) — A descrição do card.
  • imagemFundo: string (opcional) — Caminho da imagem de fundo do card. Um gradiente escuro é aplicado automaticamente para garantir contraste. Se não informado, será usada a imagem padrão do componente.
  • id: string (opcional) — ID único do card para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
  • acao: EventEmitter<void> (opcional) — Evento emitido ao clicar no card.
Instalação:
npm install mpc-lib-angular
Importação no arquivo .ts:

import { MpcCardBackGroundImgComponent } from 'mpc-lib-angular';

// No seu componente
imports: [MpcCardBackGroundImgComponent]
Exemplo de uso no arquivo .html:
<!-- Card com Background customizado -->
<mpc-card-background-img
  titulo="Programação"
  subtitulo="Horários dos Cultos"
  descricao="Confira nossa programação semanal"
  id="card-programacao"
  [tabIndex]="0"
  imagemFundo="/assets/img/exemplo.jpg"
  ariaLabel="Card de programação semanal">
</mpc-card-background-img>

<!-- Card com Background padrão (sem imagemFundo) -->
<mpc-card-background-img
  titulo="Sem imagem customizada"
  descricao="Usa a imagem padrão do componente"
  id="card-sem-img"
/>
Observação: O componente aplica automaticamente um gradiente escuro sobre a imagem de fundo para garantir a legibilidade do texto. Caso a propriedade imagemFundo não seja informada, será utilizada a imagem padrão /assets/img/no-image.jpg.

Cards de Menu

Descrição: O mpc-card-menu é um componente para exibir cards de menu interativos com ícone, título, descrição e ação de clique.
Propriedades:
  • titulo: string (obrigatório) — O título principal do card.
  • descricao: string (opcional) — A descrição do card.
  • icone: string (obrigatório) — Classe do ícone (ex: 'bi bi-telephone').
  • id: string (opcional) — ID único do card para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
  • acao: EventEmitter<void> (opcional) — Evento emitido ao clicar no card.
Instalação:
npm install mpc-lib-angular
Importação no arquivo .ts:

import & MpcCardMenuComponent } from 'mpc-lib-angular';

// No seu componente
imports: [MpcCardMenuComponent]
Exemplo de uso no arquivo .html:
<!-- Card de Menu Básico -->
<mpc-card-menu
  icone="bi bi-telephone"
  titulo="Telefone"
  descricao="(63) 9 9201-4337"
  (acao)="onMenuCardClick()">
</mpc-card-menu>

<!-- Card de Menu com Acessibilidade -->
<mpc-card-menu
  icone="bi bi-github"
  titulo="GitHub"
  descricao="Veja nossos projetos"
  id="menu-github"
  [tabIndex]="0"
  ariaLabel="Acessar perfil do GitHub"
  (acao)="onGitHubClick()">
</mpc-card-menu>

Cards de Evento

Descrição: O mpc-card-evento é um componente para exibir cards de eventos com design de timeline, incluindo data, título, subtítulo e descrição.
Propriedades:
  • titulo: string (obrigatório) — O título principal do evento.
  • subtitulo: string (opcional) — O subtítulo do evento.
  • descricao: string (opcional) — A descrição do evento.
  • dia: string (obrigatório) — O dia do evento.
  • mes: string (obrigatório) — O mês do evento.
  • id: string (opcional) — ID único do card para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
Instalação:
npm install mpc-lib-angular
Importação no arquivo .ts:

import { MpcCardEventoComponent } from 'mpc-lib-angular';

// No seu componente
imports: [MpcCardEventoComponent]
Exemplo de uso no arquivo .html:
<!-- Card de Evento Básico -->
<mpc-card-evento
