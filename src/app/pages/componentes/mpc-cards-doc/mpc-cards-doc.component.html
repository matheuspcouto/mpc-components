MPC Cards

Descrição: O mpc-card é um componente avançado para exibir cards personalizados com diferentes layouts, suportando conteúdo rico incluindo imagens, títulos, subtítulos, descrições e links. Oferece suporte completo a acessibilidade e responsividade.
Propriedades:
  • titulo: (obrigatório) O título principal do card.
  • subtitulo: (opcional) O subtítulo do card.
  • descricaoUm: (opcional) A primeira descrição do card.
  • descricaoDois: (opcional) A segunda descrição do card.
  • imagem: (opcional) A imagem do card. Exemplo: '/assets/img/imagem.jpg'.
  • links: (opcional) Lista de links do card. Exemplo: [{ icone: 'bi bi-linkedin', url: 'https://linkedin.com' }]
  • layout: (opcional) Layout do card: vertical ou horizontal. Padrão: vertical.
  • estilos: (opcional) Objeto com estilos personalizados para cores e aparência.
  • id: (opcional) O ID único do card para acessibilidade.
  • tabIndex: (opcional) O índice de tabulação para navegação por teclado.
  • ariaLabel: (opcional) Rótulo para leitores de tela.
Importação no arquivo .ts:
imports: [ MpcCardComponent ]
Exemplo de uso no arquivo .html:
<!-- Card Vertical Básico -->
<mpc-card
  titulo="Card 1"
  subtitulo="Layout Vertical"
  descricaoUm="Primeira descrição do card"
  descricaoDois="Segunda descrição do card"
  imagem="/assets/img/no-image.jpg"
  [links]="[{icone: 'bi bi-linkedin', url: 'https://www.linkedin.com/in/matheuspcouto/'}]"
  layout="vertical">
</mpc-card>

<!-- Card Horizontal com Links -->
<mpc-card
  titulo="Card Horizontal"
  subtitulo="Com Links"
  descricaoUm="Descrição do card horizontal"
  imagem="/assets/img/no-image.jpg"
  [links]="[
    {icone: 'bi bi-linkedin', url: 'https://linkedin.com'},
    {icone: 'bi bi-github', url: 'https://github.com'}
  ]"
  layout="horizontal">
</mpc-card>

<!-- Card com Estilos Personalizados -->
<mpc-card
  titulo="Card Personalizado"
  [estilos]="{
    background: '#f8f9fa',
    textColor: '#212529',
    borderColor: '#007bff',
    backgroundLinks: '#007bff',
    linksColor: '#ffffff'
  }">
</mpc-card>

Cards com Background

Descrição: O mpc-card-background-img é um componente avançado para exibir cards com imagem de fundo e gradiente escuro, ideal para destacar informações importantes com visual impactante. Oferece suporte completo a acessibilidade e responsividade.
Propriedades:
  • titulo: (obrigatório) O título principal do card.
  • subtitulo: (opcional) O subtítulo do card (ex: horário).
  • descricao: (opcional) A descrição do card.
  • imagemFundo: (obrigatório) A imagem de fundo do card. Exemplo: '/assets/img/home/mountain.jpg'.
  • estilos: (opcional) Objeto com estilos personalizados para cores e aparência.
  • id: (opcional) O ID único do card para acessibilidade.
  • tabIndex: (opcional) O índice de tabulação para navegação por teclado.
  • ariaLabel: (opcional) Rótulo para leitores de tela.
Importação no arquivo .ts:
imports: [ MpcCardBackGroundImgComponent ]
Exemplo de uso no arquivo .html:
<!-- Card Básico -->
<mpc-card-background-img
  titulo="Card 1"
  subtitulo="com imagem de fundo"
  descricao="Este é um card com imagem de fundo"
  imagemFundo="/assets/img/home/mountain.jpg">
</mpc-card-background-img>

<!-- Card com Cores Personalizadas -->
<mpc-card-background-img
  titulo="Evento Especial"
  subtitulo="Sábado às 20H"
  descricao="Um momento único de comunhão e adoração"
  imagemFundo="/assets/img/evento.jpg"
  [estilos]="{
    textColor: '#ffffff',
    barColor: '#ff6b35'
  }">
</mpc-card-background-img>

<!-- Card com Acessibilidade -->
<mpc-card-background-img
  titulo="Programação"
  subtitulo="Horários dos Cultos"
  descricao="Confira nossa programação semanal"
  imagemFundo="/assets/img/programacao.jpg"
  id="card-programacao"
  [tabIndex]="0"
  ariaLabel="Card de programação semanal">
</mpc-card-background-img>

Cards de Menu

Descrição: O mpc-card-menu é um componente avançado para exibir cards de menu interativos com ícone, título, descrição e ação de clique. Ideal para navegação rápida e ações de menu. O card possui design limpo com ícone circular e layout flexível.
Propriedades:
  • titulo: (obrigatório) O título principal do card.
  • descricao: (opcional) A descrição do card.
  • icone: (obrigatório) O ícone do card (classe do Bootstrap Icons).
  • estilos: (opcional) Objeto com estilos personalizados para cores e aparência.
  • acao: (opcional) Evento emitido ao clicar no card.
  • id: (opcional) O ID único do card para acessibilidade.
  • tabIndex: (opcional) O índice de tabulação para navegação por teclado.
  • ariaLabel: (opcional) Rótulo para leitores de tela.
Importação no arquivo .ts:
imports: [ MpcCardMenuComponent ]
Exemplo de uso no arquivo .html:
<!-- Card de Menu Básico -->
<mpc-card-menu
  icone="bi bi-telephone"
  titulo="Telefone"
  descricao="(63) 9 9201-4337"
  (acao)="onMenuCardClick()">
</mpc-card-menu>

<!-- Card de Menu com Estilos Personalizados -->
<mpc-card-menu
  icone="bi bi-linkedin"
  titulo="LinkedIn"
  descricao="Conecte-se conosco"
  [estilos]="{
    background: '#0077b5',
    textColor: '#ffffff',
    iconeBackground: '#ffffff',
    iconeColor: '#0077b5'
  }"
  (acao)="onLinkedInClick()">
</mpc-card-menu>

<!-- Card de Menu com Acessibilidade -->
<mpc-card-menu
  icone="bi bi-github"
  titulo="GitHub"
  descricao="Veja nossos projetos"
  id="menu-github"
  [tabIndex]="0"
  ariaLabel="Acessar perfil do GitHub"
  (acao)="onGitHubClick()">
</mpc-card-menu>

Cards de Evento

Descrição: O mpc-card-evento é um componente avançado para exibir cards de eventos com design de timeline, ideal para exibir informações de eventos com data, título, subtítulo e descrição. O card possui um layout único com círculo de data e linha de timeline.
Propriedades:
  • titulo: (obrigatório) O título principal do evento.
  • subtitulo: (opcional) O subtítulo do evento.
  • descricao: (opcional) A descrição do evento.
  • dia: (obrigatório) O dia do evento.
  • mes: (obrigatório) O mês do evento.
  • estilos: (opcional) Objeto com estilos personalizados para cores e aparência.
  • id: (opcional) O ID único do card para acessibilidade.
  • tabIndex: (opcional) O índice de tabulação para navegação por teclado.
  • ariaLabel: (opcional) Rótulo para leitores de tela.
Importação no arquivo .ts:
imports: [ MpcCardEventoComponent ]
Exemplo de uso no arquivo .html:
<!-- Card de Evento Básico -->
<mpc-card-evento
  titulo="Culto de Adoração"
  subtitulo="Domingo"
  descricao="Venha nos conhecer e traga sua família"
  dia="15"
  mes="Jun">
</mpc-card-evento>

</mpc-card-evento>

<!-- Card de Evento com Acessibilidade -->
<mpc-card-evento

</mpc-card-evento>
