Descrição: O mpc-tabs é um componente que exibe uma lista de abas (tabs) com títulos e conteúdos customizáveis, permitindo seleção e acessibilidade.
Propriedades do Componente:
  • tabs: Tab[] (obrigatório) — Lista de abas a serem exibidas.
  • id: string (opcional) — ID do componente para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação do componente.
  • ariaLabel: string (opcional) — Rótulo de acessibilidade das tabs.
Instalação:
npm install mpc-lib-angular
Importação no arquivo .ts:

import { MpcTabsComponent, Tab } from 'mpc-lib-angular';

// No seu componente
imports: [ MpcTabsComponent ]
Exemplo de uso no arquivo .html:
<!-- Tabs Básicas -->
<mpc-tabs 
  [tabs]="tabs" 
  (tabSelected)="selectTab($event)">
</mpc-tabs>

<!-- Tabs com Acessibilidade -->
<mpc-tabs 
  [tabs]="tabs" 
  (tabSelected)="selectTab($event)"
  id="tabs-exemplo"
  [tabIndex]="0"
  ariaLabel="Navegação por abas">
</mpc-tabs>

<!-- Conteúdo das abas -->
@if (tabSelecionada.id === 'conteudoTab1') {
  <div>Conteúdo 1</div>
}
@if (tabSelecionada.id === 'conteudoTab2') {
  <div>Conteúdo 2</div>
}
Personalização com Variáveis CSS:
/* Cores das Tabs */
--mpc-color-text-tabs: var(--mpc-color-primary);      /* Cor do texto das tabs */
--mpc-color-bg-tabs: white;                           /* Cor de fundo das tabs */
--mpc-color-border-tabs: var(--mpc-color-primary);    /* Cor da borda das tabs */
--mpc-color-active-tabs: var(--mpc-color-primary);    /* Cor da aba ativa */
--mpc-color-hover-tabs: var(--mpc-color-secondary);   /* Cor do hover */

/* Fontes */
--mpc-font-tabs: var(--mpc-font-default);             /* Fonte das tabs */
Eventos:
  • tabSelected: Emite o objeto Tab selecionado ao trocar de aba.
Interface Tab:
  • id: string — Identificador único da aba.
  • titulo: string — Título exibido na aba.
