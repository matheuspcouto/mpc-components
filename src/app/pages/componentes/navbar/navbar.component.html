MPC Navbar

O mpc-navbar é um componente que permite a criação de um menu de navegação. Ele é composto por um menu de navegação com submenus, uma logo e um botão de ação.
Para alterar as abas e rotas do navbar, basta alterar as propriedades no mpc-navbar.component.html.
No arquivo seu-component.ts, importe o componente:
              
                ```
                imports: [ MpcNavbarComponent ]
                ```
No arquivo seu-component.html, adicione o seguinte código: 
              
              <mpc-navbar></mpc-navbar>
Navegue pelo menu desta página para ver implementação do mpc-navbar.
