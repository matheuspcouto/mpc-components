MPC Navbar

Descrição: O mpc-navbar é um componente de barra de navegação responsiva, baseado no Bootstrap, que facilita a navegação entre as páginas do sistema.
Propriedades:
  • abaLogin: (opcional) Rota para a página de login.
  • itens: (opcional) Lista de itens de navegação. Exemplo: [{ label: 'Home', rota: '/home' }]
Importação no arquivo .ts:
imports: [ MpcNavbarComponent ]
Exemplo de uso no arquivo .html:
<mpc-navbar abaLogin="/paginas/login" [itens]="[{ label: 'Home', rota: '/home' }]"></mpc-navbar>
