Propriedades do Componente:
  • id: string (obrigatório) — ID do divisor para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
  • titulo: string (obrigatório) — Título principal do divisor.
  • subtitulo: string (opcional) — Subtítulo do divisor.
  • imagemFundo: string (obrigatório) — Caminho da imagem de fundo do divisor. Um gradiente escuro é aplicado automaticamente para garantir contraste.
Importação no arquivo .ts:

import { MpcPageDividerImgComponent } from 'mpc-lib-angular';

// No seu componente
imports: [MpcPageDividerImgComponent]
Exemplo de uso no arquivo .html:
<!-- Page Divider com Imagem de Fundo -->
<mpc-page-divider-img
  titulo="Exemplo de"
  subtitulo="Page Divider"
  id="divider-exemplo"
  [tabIndex]="0"
  imagemFundo="/assets/img/home/mountain.jpg"
  ariaLabel="Exemplo" />
Observação: O componente aplica automaticamente um gradiente escuro sobre a imagem de fundo para garantir a legibilidade do texto.
Personalização com Variáveis CSS:
/* Personalize o divisor de página */
--mpc-font-title-page-divider-img: var(--mpc-font-title);
--mpc-font-subtitle-page-divider-img: var(--mpc-font-subtitle);
--mpc-color-title-page-divider-img: var(--mpc-color-primary);
--mpc-color-subtitle-page-divider-img: var(--mpc-color-secondary);
