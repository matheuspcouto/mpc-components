MPC Pagination

O mpc-pagination é um componente que permite criar paginação personalizada. A paginação é utilizada para dividir grandes conjuntos de dados em páginas menores, facilitando a navegação e melhorando a performance. Abaixo, você pode ver como utilizar o componente e seus parâmetros.
  • totalItens: O número total de itens a serem paginados.
  • mostrarSeletorItensPorPagina: (opcional) Indica se deve mostrar o seletor de itens por página. Padrão: true.
  • tabIndex: (opcional) O tabIndex do componente.
  • ariaLabel: (opcional) O texto de acessibilidade do componente.
  • indices: Evento emitido quando a página é alterada, retorna os índices de início e fim.
No arquivo seu-component.ts, importe o componente:
              
                ```
                imports: [ MpcPaginationComponent ]
                ```
Ainda no seu-component.ts, crie as propriedades necessárias para controlar a paginação e implemente o método para capturar os índices.
                
                  // Lista completa de itens
                  itensTotais: any[] = [
                    // seus dados aqui...
                  ];

                  // Lista filtrada para exibição
                  itensFiltrados: any[] = [];

                  // Método para definir os índices da paginação
                  definirIndiceLista(indices: { inicio: number, fim: number }) {
                    this.itensFiltrados = this.itensTotais.slice(indices.inicio, indices.fim);
                  }
No arquivo seu-component.html, adicione o seguinte código: 
                
                  <mpc-pagination
                    [totalItens]="itensTotais.length"
                    [mostrarSeletorItensPorPagina]="true"
                    [itensPorPagina]="10"
                    [tabIndex]="0"
                    ariaLabel="Paginação da lista"
                    (indices)="definirIndiceLista($event)">
                  </mpc-pagination>
