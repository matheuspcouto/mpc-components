MPC Footer

Descrição: O mpc-footer é um componente que exibe o rodapé da página, com informações sobre a empresa ou organização, links úteis e informações de contato.
Propriedades do Componente:
  • id: string (opcional) — ID do footer para acessibilidade.
  • tabIndex: number (opcional) — Índice de tabulação para navegação por teclado.
  • ariaLabel: string (opcional) — Rótulo para leitores de tela.
O componente não possui outros @Input(). É apenas visual e de acessibilidade.
Instalação:
npm install mpc-lib-angular
Importação no arquivo .ts:

import { MpcFooterComponent } from 'mpc-lib-angular';

// No seu componente
imports: [ MpcFooterComponent ]
Exemplo de uso no arquivo .html:
<!-- Footer Básico -->
<mpc-footer></mpc-footer>

<!-- Footer com Acessibilidade -->
<mpc-footer 
  id="footer-principal"
  [tabIndex]="0"
  ariaLabel="Rodapé da página principal">
</mpc-footer>
Personalização com Variáveis CSS:
/* Cores do Footer */
--mpc-color-bg-footer: var(--mpc-color-primary);      /* Cor de fundo do footer */
--mpc-color-text-footer: white;                        /* Cor do texto do footer */
--mpc-color-links-footer: white;                       /* Cor dos links do footer */
--mpc-color-hover-footer: var(--mpc-color-secondary);  /* Cor do hover dos links */

/* Fontes */
--mpc-font-footer: var(--mpc-font-default);            /* Fonte do footer */
Como personalizar:
  • Para alterar as informações do rodapé, edite o arquivo mpc-footer.component.html
  • Para usar em todas as páginas, inclua no app.component.html
  • Para personalizar cores e fontes, use as variáveis CSS acima
