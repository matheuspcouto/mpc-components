Descrição: O <mpc-error> é um componente responsável por exibir uma tela de erro completa e responsiva, com informações detalhadas, botões de ação e suporte a imagens personalizadas. Utiliza Signal para gerenciar o estado do erro automaticamente através do MpcErrorService.
Propriedades do Componente:
  • Gerenciamento de Estado: Utiliza Signal para compartilhar o estado do erro entre service e componente.
  • Exibição Automática: A tela é exibida automaticamente quando um erro é construído pelo MpcErrorService.
  • Imagens Personalizadas: Suporte a imagens personalizadas para diferentes tipos de erro.
  • Navegação: O botão "Voltar" redireciona para a rota especificada no objeto de erro.
  • Cópia de Mensagem: O botão "Copiar Mensagem" copia o título e descrição do erro para a área de transferência.
  • Responsivo: A tela se adapta a diferentes tamanhos de dispositivo.
O componente não possui @Input() diretos. Toda a configuração é feita via MpcErrorService.
⚠️ DEPENDÊNCIA:
O mpc-error utiliza o componente mpc-button da biblioteca mpc-lib-angular.
É necessário ter a mpc-lib-angular instalada no seu projeto para o funcionamento correto dos botões!
Instale com npm install mpc-lib-angular se ainda não estiver no seu projeto.
Importação no arquivo .ts:

import { MpcErrorComponent } from './shared/components/mpc-error/mpc-error.component';
import { MpcErrorService } from './shared/components/mpc-error/mpc-error.service';

// No seu componente
imports: [ MpcErrorComponent ]

// Injeção do serviço
private mpcErrorService = inject(MpcErrorService);
Exemplo de uso no arquivo .html:
<!-- O componente será exibido automaticamente quando um erro for construído pelo MpcErrorService -->
<mpc-error></mpc-error>
Exemplo de uso do MpcErrorService no arquivo .ts:

// Com imagem padrão
mpcErrorService.construirErro(erro, rotaRetorno);

// Com imagem personalizada
mpcErrorService.construirErro(erro, rotaRetorno, 'caminho/para/imagem.png');
Personalização com Variáveis CSS:
/* Personalize as cores e imagens do erro conforme sua necessidade. */
/* Exemplo de variáveis (adicione no seu CSS global se desejar): */
--mpc-error-bg: #fff3f3; /* Cor de fundo do erro */
--mpc-error-title: #d32f2f; /* Cor do título do erro */
--mpc-error-description: #333; /* Cor da descrição */
--mpc-error-btn: var(--mpc-color-primary); /* Cor dos botões */
Como personalizar:
  • Para alterar as mensagens de erro, utilize o MpcErrorService no seu componente.
  • Para personalizar imagens, passe o caminho da imagem ao construir o erro.
  • Para alterar cores e estilos, utilize as variáveis CSS acima ou edite o arquivo mpc-error.component.css
