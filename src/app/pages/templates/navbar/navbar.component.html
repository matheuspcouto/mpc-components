Descrição: O <navbar> é um componente de navegação estilizado e acessível para uso em aplicações Angular, permitindo abas, submenus e identificação automática de login.
Propriedades do Componente:
  • [abasInput]: NavbarConfig[] (obrigatório) — Array de configuração das abas da navbar.
O componente não possui outros @Input(). Toda a navegação é definida via abasInput.
⚠️ IMPORTANTE:
Este componente NÃO faz parte da biblioteca mpc-lib-angular!
Para utilizá-lo, você precisa:
  1. Baixar o projeto do GitHub
  2. Copiar a pasta src/app/shared/components/mpc-navbar/ para o seu projeto
  3. Importar o componente diretamente no seu módulo/componente
Importação no arquivo .ts:

import { MpcNavbarComponent } from './shared/components/mpc-navbar/mpc-navbar.component';

// No seu componente
imports: [ MpcNavbarComponent ]
Exemplo de uso no arquivo .html:
<!-- Navbar Básica -->
<mpc-navbar [abasInput]="abas"></mpc-navbar>

<!-- Exemplo de configuração das abas no arquivo .ts -->
const abas: NavbarConfig[] = [
  {
    titulo: 'Home',
    rota: '/',
    icone: 'bi bi-house'
  },
  {
    titulo: 'Perfil',
    rota: '/perfil',
    icone: 'bi bi-person',
    subRotas: [
      { titulo: 'Dados Pessoais', rota: '/perfil/dados' },
      { titulo: 'Configurações', rota: '/perfil/config' }
    ]
  },
  {
    titulo: 'Login',
    rota: '/login',
    icone: 'bi bi-box-arrow-in-right'
  }
];
Personalização com Variáveis CSS:
/* Cores da Navbar */
--mpc-color-bg-header: white;                  /* Cor de fundo da navbar */
--mpc-color-text-disabled: #7f7f90;            /* Cor do texto desabilitado */
--mpc-color-text-active: var(--mpc-color-primary); /* Cor do texto ativo */
--mpc-color-text-hover: var(--mpc-color-primary);  /* Cor do texto no hover */
--mpc-color-border-active: var(--mpc-color-secondary); /* Cor da borda ativa */

/* Fontes */
--mpc-font-navbar: var(--mpc-font-subtitle);   /* Fonte da navbar */
Como personalizar:
  • Para alterar as abas, edite o array abasInput no seu arquivo .ts
  • Para usar em todas as páginas, inclua no app.component.html
  • Para personalizar cores e fontes, use as variáveis CSS acima
