MPC Library Angular

Descrição: A mpc-lib-angular é uma biblioteca de componentes Angular com sistema de design customizável, desenvolvida para facilitar a criação de interfaces modernas e responsivas. A biblioteca oferece componentes prontos para uso com estilos personalizáveis através de variáveis CSS.

Requisitos do Sistema

Versões Suportadas:
  • Node.js: 20.x ou superior
  • Angular: 19.x ou superior
  • Bootstrap: 5.x
Dependências:
  • Bootstrap Icons: Para ícones dos componentes
  • Bootstrap CSS: Para estilos base
  • Angular Common: Para funcionalidades básicas

Instalação

1. Instalar a Biblioteca:
npm install mpc-lib-angular
2. Instalar Bootstrap (se ainda não tiver):
npm install bootstrap bootstrap-icons
3. Configurar Bootstrap no angular.json:
{
  "styles": [
    "node_modules/bootstrap/dist/css/bootstrap.min.css",
    "node_modules/bootstrap-icons/font/bootstrap-icons.css",
    "node_modules/mpc-lib-angular/lib/styles/mpc-lib-angular.css",
    "src/styles.css"
  ]
}

Configuração dos Estilos

Importar os Estilos da Biblioteca:

No seu arquivo angular.json, adicione os estilos da biblioteca:

{
  "styles": [
    "node_modules/mpc-lib-angular/lib/styles/mpc-lib-angular.css",
    "src/styles.css"
  ]
}
Ou no seu styles.css:
@import '~mpc-lib-angular/lib/styles/mpc-lib.css';

Variáveis CSS Disponíveis

Fontes:
--mpc-font-title: 'Potential', sans-serif;
--mpc-font-default: "Poppins", sans-serif;
--mpc-font-subtitle: "Inter", sans-serif;
Cores:
--mpc-color-primary: #0E2749;
--mpc-color-secondary: #149DDD;
--mpc-color-tertiary: gray;

Como Sobrescrever Variáveis CSS

Opção 1: Sobrescrever no :root (Recomendado)

No seu arquivo styles.css:

:root {
  /* Sobrescrever cores da biblioteca */
  --mpc-color-primary: #your-custom-color;
  --mpc-color-secondary: #your-custom-secondary;
  --mpc-color-tertiary: #your-custom-color;
  
  /* Sobrescrever fontes */
  --mpc-font-default: "Your Custom Font", sans-serif;
  --mpc-font-title: 'AAA', sans-serif;
  --mpc-font-subtitle: "BBB", sans-serif;
}
Opção 2: Sobrescrever em um Container Específico
.my-custom-theme {
  --mpc-color-primary: #your-custom-color;
  --mpc-color-secondary: #your-custom-secondary;
  --mpc-color-tertiary: #your-custom-color;
}
Opção 3: Usando CSS Custom Properties com Fallbacks
:root {
  --mpc-color-primary: var(--your-custom-primary, #0E2749);
  --mpc-color-secondary: var(--your-custom-secondary, #149DDD);
}

Exemplo de Uso

HTML com Classes Utilitárias:
<div class="mpc-bg-primary">
  <h1 class="mpc-text-primary">Título</h1>
  <p class="mpc-text-secondary">Descrição</p>
</div>
Importação de Componentes:
import { MpcCardComponent } from 'mpc-lib-angular';

@Component({
  selector: 'app-example',
  imports: [MpcCardComponent],
  templateUrl: './example.component.html'
})
export class ExampleComponent {
  // Seu código aqui
}

Componentes Disponíveis

Cards:

  • MpcCardBackGroundImgComponent - Cards com imagem de fundo
  • MpcCardMenuComponent - Cards de menu
  • MpcCardEventoComponent - Cards de evento
Inputs:
  • MpcInputTextComponent - Input de texto
  • MpcInputEmailComponent - Input de email
  • MpcInputSenhaComponent - Input de senha
  • MpcInputTelefoneComponent - Input de telefone
  • MpcInputCpfCnpjComponent - Input de CPF/CNPJ
  • MpcInputDateComponent - Input de data

  • MpcInputSelectComponent - Input select
  • MpcInputRadioComponent - Input radio
