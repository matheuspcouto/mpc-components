MPC Error Component

O componente mpc-error é responsável por exibir uma tela de erro completa com informações detalhadas. Utiliza Signal para gerenciar o estado do erro automaticamente através do ErrorService. A tela é exibida quando um erro é construído pelo service e oferece duas opções: voltar para uma rota específica ou copiar a mensagem de erro para a área de transferência. Suporta imagens personalizadas para diferentes tipos de erro.
  • Gerenciamento de Estado: Utiliza Signal para compartilhar o estado do erro entre service e componente.
  • Exibição Automática: A tela é exibida automaticamente quando um erro é construído pelo ErrorService.
  • Imagens Personalizadas: Suporte a imagens personalizadas para diferentes tipos de erro.
  • Navegação: O botão "Voltar" redireciona para a rota especificada no objeto de erro.
  • Cópia de Mensagem: O botão "Copiar Mensagem" copia o título e descrição do erro para a área de transferência.
  • Responsivo: A tela se adapta a diferentes tamanhos de dispositivo.
Para usar o componente, importe e use o ErrorService:
              
                ```
                // No componente
                private errorService = inject(ErrorService);
                
                // Com imagem padrão
                this.errorService.construirErro(erro, rotaRetorno);
                
                // Com imagem personalizada
                this.errorService.construirErro(erro, rotaRetorno, 'caminho/para/imagem.png');
                ```
